# Тест поддержки GIF анимации

## Постановка задачи
Проверить возможность разметки GIF-анимации в обеих платформах.

## Результаты тестирования

### CVAT
**Нативная поддержка GIF**: ❌ Нет

**Решение**: Конвертация в MP4
- Использован ffmpeg для конвертации
- Команда: `ffmpeg -i animation.gif -movflags faststart -pix_fmt yuv420p output.mp4`
- Результат: успешная загрузка и разметка

**Процесс разметки:**
- GIF конвертирован в MP4 без потери качества
- Все кадры сохранены
- Применены стандартные инструменты разметки видео
- Трекинг работает корректно

### Label Studio  
**Нативная поддержка GIF**: ❌ Нет

**Решение**: Конвертация в MP4
- Аналогичная конвертация через ffmpeg
- Загрузка как обычного видео
- Покадровая разметка

**Процесс разметки:**
- Конвертированное видео загружено успешно
- Разметка через видеоплеер
- Базовая интерполяция между кадрами

## Сравнение подходов

| Параметр | CVAT | Label Studio |
|----------|------|--------------|
| **Конвертация** | Требуется | Требуется |
| **Качество после конвертации** | Отличное | Отличное |
| **Удобство разметки** | Высокое | Среднее |
| **Сохранение анимации** | Да | Да |
| **Время подготовки** | 2-3 минуты | 2-3 минуты |

## Практические рекомендации

**Для работы с GIF в обеих платформах:**

1. **Предварительная подготовка:**
   - Конвертировать GIF в MP4/AVI
   - Проверить frame rate и качество
   - Убедиться в сохранении всех кадров

2. **Оптимальные настройки конвертации:**
   ```bash
   ffmpeg -i input.gif -movflags faststart -pix_fmt yuv420p -r 30 output.mp4
   ```

3. **Workflow:**
   - CVAT: загрузить как видео → использовать трекинг
   - Label Studio: загрузить как видео → ручная разметка

## Вывод
Обе платформы требуют предварительной конвертации, но CVAT лучше подходит для сложной разметки благодаря трекингу.
